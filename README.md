# 🎥 AttentionX - Automated Content Repurposing Engine

## 🧠 Problem Statement
Creators produce long-form video content such as lectures, podcasts, and workshops. However, modern audiences prefer short-form content. Important insights are often buried inside long videos, making them hard to access.

## 💡 Solution
AttentionX is a GenAI-powered system that processes long-form videos and extracts high-impact highlights.

The system converts video into text and identifies important segments, making content more accessible and engaging.

## ⚙️ Features
- Upload long-form video
- Automatic speech-to-text transcription
- Highlight extraction using scoring logic
- Simple and interactive UI

## 🏗️ Tech Stack
- Python
- Streamlit
- OpenAI Whisper

## 🔄 How It Works
1. Upload a video
2. Convert speech to text using Whisper
3. Split transcript into sentences
4. Score sentences based on importance
5. Display top highlights

## 🚀 Future Scope
- Clip generation using MoviePy
- Advanced semantic understanding using Gemini API
- Emotion detection using Librosa
- Smart cropping using MediaPipe
